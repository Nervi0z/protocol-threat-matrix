# Security Policy

## Scope

This repository contains documentation, protocol references, detection indicators, and ATT&CK mappings. It does not include executable code or deployed infrastructure.

## Reporting

If you find an inaccurate detection claim that could cause a false sense of security, a factual error in an ATT&CK mapping, or any other security concern with the documentation — contact privately:

**nervi-zz@protonmail.com**

Include: description of the problem, its location, and potential impact.

## Usage notice

All detection techniques and protocol analysis documented here are for authorized defensive use — monitoring your own infrastructure or systems you have explicit permission to analyze.
